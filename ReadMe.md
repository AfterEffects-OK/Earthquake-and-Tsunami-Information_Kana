# 地震情報ライブモニター (Earthquake Information Live Monitor)

地震・津波情報をリアルタイムで表示するためのWebアプリケーションです。
P2P地震情報APIから取得したデータを、配信用途に適した形式で画面に表示します。

## 主な機能

- **リアルタイム地震情報表示**:
  - P2P地震情報API v2を利用して、最新の地震情報（震源・震度情報）を取得します。
  - 設定した震度以上の地震をリスト形式で表示します。

- **詳細情報パネル**:
  - 選択した地震の震源地、発生時刻、規模（マグニチュード）、深さを表示します。
  - 震度別に観測地点を一覧表示します。
  - 表示モードを「市区町村別（ふりがな付き）」と「観測点名」で切り替え可能です。

- **津波情報**:
  - 大津波警報、津波警報、津波注意報の発表状況と対象エリアを表示します。
  - 各地の津波観測点で観測された津波の高さ（観測情報）を表示します。

- **配信用固定テロップ**:
  - 画面下部に、配信画面への合成を想定した固定テロップを表示します。
  - 地震の概況、津波情報、震度別の地域情報を自動でページ送り再生（ループ再生対応）できます。
  - 再生時間、ループ回数、表示する最低震度などを設定可能です。

- **訓練モード**:
  - 過去の大地震（東日本大震災など）や、南海トラフ巨大地震などの想定データをダミーデータとして読み込み、表示や再生のテストができます。

- **データ記録**:
  - 取得した地震情報と観測点情報をGoogleスプレッドシートに自動で記録します（要Google Apps Script設定）。

## セットアップと設定

このアプリケーションを使用するには、いくつかの初期設定が必要です。
`script.js`ファイルを開き、以下の項目を設定してください。

### 1. Google Apps Script ウェブアプリURLの設定

地震情報と観測点情報をGoogleスプレッドシートに記録するために、Google Apps Scriptで作成したウェブアプリのURLを設定します。

```javascript
// script.js

const CONFIG = {
    // ...（他の設定）

    // 【要設定】Google Apps ScriptのウェブアプリURL
    GAS_WEB_APP_URL: 'ここに地震情報記録用のURLを貼り付け',

    // 【要設定】観測点情報記録用のGoogle Apps ScriptウェブアプリURL
    GAS_POINTS_WEB_APP_URL: 'ここに観測点情報記録用のURLを貼り付け',
};
